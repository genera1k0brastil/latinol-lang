# Gramer / Grammar

## 1. Sesler (Phonology)

| Harf | Ses | Örnek |
|------|-----|-------|
| a | /a/ | aqua |
| e | /e/ | esse |
| i | /i/ | ignis |
| o | /o/ | homo |
| u | /u/ | unus |
| c | /k/ | semper |
| v | /v/ | ventus |
| rr | /r̄/ | carrus |
| z | /ts/ | zephyrus |

## 2. Çoğul Yapımı

| Sonuç | Çoğul | Örnek |
|-------|-------|-------|
| -a | -as | luna → lunas |
| -o, -u | -os | homo → homos |
| Ünsüz | -es | mons → monses |

## 3. Fiil Çekimi

Şahıs zamiri zorunludur. Fiil sadece tekil/çoğul ayrılır.

| Zaman | Tekil | Çoğul | Örnek (ama) |
|-------|-------|-------|-------------|
| Şimdiki | -a | -an | eo ama / nos aman |
| Geçmiş | -ava | -avan | eo amava |
| Gelecek | -ara | -aran | eo amara |

## 4. Cümle Sırası

**SOV:** Özne — Nesne — Fiil

&gt; homo lunam ama → Adam ayı sever.

## 5. Olumsuzluk

Tek olumsuz: **no** (fiilin önüne gelir)

&gt; eo terram no ama → Ben toprağı sevmem.
