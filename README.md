# Latinol

Latince · İspanyolca Hibrit Yapay Dil  
*A constructed language blending Latin and Spanish roots.*

## Hızlı Başlangıç (Quick Start)

**Merhaba:** Salve!  
**Nasılsın:** Quomodo tu esa?  
**Teşekkürler:** Gratias!

## Temel Özellikler
- Cinsiyetsiz gramer
- Sadeleştirilmiş fiil çekimi (2 tekil/çoğul form)
- SOV cümle sırası
